# Usage

1. NeoBundleインストール
```
 $ curl https://raw.githubusercontent.com/Shougo/neobundle.vim/master/bin/install.sh > install.sh
 $ sh ./install.sh
($ mkdir ~/.vim/bundle)
 ↓
:NeoBundleInstall
```

2. シンボリックリンクで繋げる
```
$ ln -s ~/setting/vimrc/_vimrc ~/.vimrc
$ ln -s ~/setting/indent ~/.vim/indent
$ ln -s ~/setting/indent ~/.vim/indent
$ ln -s ~/setting/colors ~/.vim/colors
```
